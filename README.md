# Minimal Web App Template

A lightweight Next.js web app with local storage for data persistence.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Features

- Next.js 14
- TypeScript
- Local Storage integration
- Fully client-side
- Minimal dependencies
- Fast build times

## How LocalStorage Works

Data is automatically saved to browser's localStorage and persists across page reloads. No server required!
